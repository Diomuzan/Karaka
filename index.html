Toets JavaScript periode 4

Welkom bij de JavaScript toets. Er zijn 5 opdrachten, voor elke opdracht lees je goed wat de bedoeling is op die pagina.
Als je een variabele maakt kies jezelf voor let, var of const. Let op dat je de juiste keuze maakt die past bij de opdracht/ code.
Je mag het internet gebruiken bij deze toets, maar geen chatapplicaties zoals Microsoft Teams.
Je kan 55 punten voor deze toets halen. 50 voor de opdrachten en 5 voor de netheid/ correctheid van jouw code.
Jouw code wordt ook beoordeeld op de volgende punten:
  • Netheid van de code - 1 punten
  • Correct indenten van de code - 1 punten
  • Correct gebruik en positie van { } [ ] tekens in jouw code - 1 punten
  • Gebruik van camelCase bij functies en variabelen benamingen - 1 punten
  • Engelse taal van functies en variabelen benamingen - 1 punten

Er is nog geen Javascript bestand

Geen enkel HTML-bestand heeft nu een JavaScript bestand gekoppeld. Zorg ervoor dat de HTML-pagina's assigment-1.html t/m assignment-5.html ieder een eigen Javascript bestand gekoppeld krijgen.

Er komen dus 5 losse Javascript bestanden in de juiste map te staan.

Zorg ervoor dat elk Javascript bestand begint met een melding in de console: ‘assignment-X’ wordt goed ingeladen.

Vervang hier de ‘x’ steeds door het juiste nummer van de opdracht.
